Filesystem Hierarchy

./ TOP LEVEL ROOT DIRECTORY
./ROOT   ROOT USERS HOME DIRECTORY
./HOME   HOME DIRECTORY FOR OTHER USERS
./BOOT   ITS CONTAINES BUTABLE  FILES FOR LINUX
./ETC   ITS CONTAINES   ALL CONFIGURATION FAILS
./USR   BY DEFAULT SOFTWARE AND INSTALLERS INSTALLED IN THIS DIRECTORY
./BIN   ITS CONTAINES COMMANDS USED ALL USERS
./SBIN  ITS CONTAINES COMMANDS USED ONLY ROOT USERS
